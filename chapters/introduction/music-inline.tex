\section{Inline music (single staff, paragraph-sized)}

This is the smallest ``music inside a sentence'' unit: one staff, paragraph scale, no apology for being technical because the reader already agreed to read.

Inline excerpts use a tighter staff size so a short gesture can sit beside text without dominating the page. The sample cell on this page is pulled in from its own fragment so a copy editor and an engraver can work in parallel.
